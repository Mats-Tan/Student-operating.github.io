win10 不显示高性能模式时切换教程

三行命令解决

Win + R 输入 cmd ，打开命令行窗口

  1. 输入 powercfg/L 查看当前电源模式

  2. 输入 powercfg/ALIASES 查看电脑拥有的电源方案

  3. 找到和自己电源模式名字相似，后缀为 MIN 的方案（ 我的为 SCHEME_MIN )

    输入 powercfg/S SCHEME_MIN ，再打开其他电源设置，发现修改成功。（或输入第一条命令查看）
